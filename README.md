# 📘 Manual Book
## Sistem Peminjaman Alat Sarana Sekolah

---

## 1. Deskripsi Sistem
Aplikasi Peminjaman Alat Berbasis Web merupakan sistem yang dirancang untuk mempermudah proses pengajuan dan pengelolaan peminjaman alat secara digital. Sistem ini menggantikan proses manual yang sering menimbulkan masalah seperti kehilangan data, kesalahan pencatatan, dan kesulitan dalam pengecekan ketersediaan alat.

Aplikasi ini memiliki tiga jenis pengguna:
- Admin
- Petugas
- User (Siswa/Guru)

---

## 2. Kelebihan Sistem
- Mempermudah proses peminjaman alat
- Mengurangi kesalahan pencatatan manual
- Mempermudah pengecekan ketersediaan alat
- Data tersimpan rapi dan mudah diakses
- Proses verifikasi lebih cepat

---

## 3. Kekurangan Sistem
- Bergantung pada koneksi internet
- Membutuhkan perangkat (HP/Laptop)
- Perlu maintenance sistem

---

## 4. Halaman Login & Register

### 🔐 Register
Digunakan untuk membuat akun baru (Siswa & Guru).

**Fitur:**
- Input Nama
- Pilih Status (Siswa/Guru)
- Input NISN & Kelas (Siswa)
- Input NIP (Guru)
- Email & Password
- Tombol Register & Login

**Cara penggunaan:**
1. Isi data lengkap
2. Pilih status
3. Klik Register
4. Login menggunakan akun

---

### 🔐 Login
Digunakan untuk masuk ke sistem.

**Fitur:**
- Input Email & Password
- Pesan error
- Tombol Login & Register

**Cara penggunaan:**
1. Masukkan email & password
2. Klik Login
3. Masuk ke dashboard
